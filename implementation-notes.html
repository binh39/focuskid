FocusKid ADHD/HCI redesign notes

Decisions not specified in the original request

  • Chọn hướng thiết kế Calm Adventure cho trẻ 8-12 tuổi: vui nhưng ít nhiễu, dùng màu pastel dịu và một hành động chính trên mỗi màn.
  • Đặt thời lượng tập trung mặc định là 15 phút thay vì 25 phút để phù hợp hơn với trẻ ADHD.
  • Thêm focusHelperEnabled vào preferences để camera helper có thể tắt/bật thay vì luôn tạo cảm giác bị theo dõi.
  • Đổi camera từ tự bật sang opt-in: trẻ chủ động bấm Start helper sau khi đọc giải thích quyền riêng tư.
  • Đổi popup mất tập trung từ cảnh báo đỏ sang dialog nghỉ nhẹ, dùng ngôn ngữ khuyến khích thay vì trách phạt.

Changes from the existing code

  • Thêm utility preferences có kiểm thử để lưu cài đặt tập trung vào localStorage.
  • Chuyển Settings từ UI local-only thành cài đặt có tác dụng cho timer/camera/sound reminder.
  • Thiết kế lại Child Dashboard thành màn một nhiệm vụ chính hôm nay, giảm cụm thống kê và sidebar kiểu người lớn.
  • Làm mềm Reader focus helper: camera optional, có safety copy, nút stop, và popup pause nhẹ.
  • Thêm CSS Calm Adventure với nền pastel, card bo tròn, màu mint/cream/lavender và progress semantic.

Tradeoffs

  • Dùng localStorage thay vì backend để hoàn thành nhanh trong phạm vi frontend/HCI prototype; dữ liệu đủ ổn cho demo nhưng chưa đồng bộ đa thiết bị.
  • Giữ cấu trúc file hiện tại để thay đổi tập trung, tránh refactor lớn ngoài phạm vi UI/UX.
  • Chưa làm hệ thống i18n đầy đủ; copy mới ưu tiên tiếng Anh nhất quán với phần lớn UI hiện có.

Change log by file/folder

  • src/utils/preferences.ts: thêm model cài đặt tập trung, default ADHD-friendly, clamp thời lượng an toàn và hàm load/save.
  • src/utils/preferences.test.ts: thêm Vitest tests chứng minh default, clamp và persistence hoạt động.
  • src/components/SettingsView.tsx: thiết kế lại settings thành Focus rhythm, Gentle support, Calm plan và Safety note; thêm switch ARIA.
  • src/pages/ChildDashboard.tsx: chuyển dashboard trẻ thành Calm Adventure với nhiệm vụ chính, CTA lớn, rank orb, reset plan và reward path.
  • src/components/FocusCameraPanel.tsx: bỏ auto-start camera, thêm consent UI, stop button, trạng thái helper off và copy riêng tư.
  • src/pages/ChildReader.tsx: giảm âm lượng cue, dùng preferences, popup pause nhẹ có role dialog/focus, và không resume timer khi helper vẫn đang báo mất tập trung.
  • src/assets/dashboard.css: thêm style Calm Adventure cho child dashboard.
  • src/assets/settings.css: thêm style pastel/accessible cho settings mới.
  • src/assets/reader.css: thêm style cho pause dialog và camera consent.
  • package.json/package-lock.json: thêm Vitest làm dev dependency để kiểm thử frontend utility.
